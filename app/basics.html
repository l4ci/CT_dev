Icons

Font Awesome 4 is included. There is also the option to toggle icons on click.

Just a normal icon with the class="toggle" and a data-toggle attribute with the icon to change to and any other class than fa or toggle.

Like this one: <i class="fa fa-plus toggle fa-2x" data-toggle="fa-minus fa-2x"></i>

Usage:

Icons are used from various extensions, like dropdown, mobile navigation, settings. If you modify or use another icon font, you got to modify the source code as well.
